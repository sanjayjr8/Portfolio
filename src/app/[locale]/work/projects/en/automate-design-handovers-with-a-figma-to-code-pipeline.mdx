## Overview

Development of Opti Stock, a supply chain optimization system featuring secure authentication for companies, inventory management, and predictive analytics for historical sales.

## Key Features

- **Authenticated Login**: Implemented a secure login and registration system, allowing companies to register, manage their profiles, and access the platform safely.
- **Inventory Management**: Companies can add and maintain their inventory through a user-friendly interface, ensuring up-to-date stock levels and accurate records.
- **Inventory Database**: Utilized MongoDB Atlas to create a robust inventory database that stores detailed information about products and stock levels.
- **Analytics and Reporting**: Integrated inventory analytics to provide insights into product performance, helping companies make informed business decisions.
- **Sales Prediction**: Deployed an ARIMA machine learning model via Streamlit for historical sales prediction, enabling companies to forecast demand and manage stock accordingly.
- **Frontend and Backend Technologies**: Developed the frontend using React and TypeScript, with a Node.js backend to create a seamless and efficient user experience.

## Technologies Used

- **Angular**: For building a responsive and dynamic user interface.
- **Node.js**: For backend development, handling authentication, and managing server-side logic.
- **React and TypeScript**: To create a modern, component-based frontend experience.
- **MongoDB Atlas**: To store and manage inventory data securely in the cloud.
- **ARIMA Model**: For historical sales prediction, deployed using Streamlit for easy accessibility and visualization.

## Challenges and Learnings

One key challenge was ensuring the system could efficiently handle multiple users while maintaining data integrity and security. This was addressed by implementing robust authentication mechanisms and optimizing database queries. Additionally, integrating the ARIMA model for accurate sales forecasting required extensive testing and calibration, leading to a better understanding of machine learning applications in inventory management.

## Outcome

Opti Stock is now a comprehensive supply chain optimization tool that enables companies to manage their inventory efficiently while gaining valuable insights into sales trends. The combination of secured login, real-time inventory analytics, and predictive capabilities ensures companies can make data-driven decisions, improving their overall supply chain management. The system's architecture and technology stack make it scalable, allowing for future enhancements and feature additions.
